# Task Documentation

This directory contains detailed task breakdowns for ongoing and upcoming work.

## Directory Purpose

- **Active Tasks**: Current sprint tasks awaiting implementation
- **Archived Tasks**: Historical task documentation moved to `../archived-sprints/`

## Naming Convention

Task files follow the pattern: `SP-XXX-YYY-description.md`

- `SP-XXX`: Sprint number
- `YYY`: Task sequence number within the sprint
- `description`: Brief description of task content

## Current Sprint

**Status**: Determine current sprint based on project needs

For the latest compliance status and priorities, see:
- `../roadmap/CURRENT-COMPLIANCE-STATUS.md` - Latest compliance metrics
- `../archived-sprints/` - Historical task archives
